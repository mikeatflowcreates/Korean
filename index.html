syllable block logic
초성 + 중성 + 종성
initial consonant + vowel + final consonant (optional)
Every Korean syllable is one block. You type left to right — the system stacks automatically.
Like Cantonese syllable structure: onset + nucleus + coda.
먹 = ㅁ(m) + ㅓ(eo) + ㄱ(k) · 한 = ㅎ(h) + ㅏ(a) + ㄴ(n)
받침 = the bottom consonant. Its presence determines 이에요 vs 예요.
